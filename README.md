# Preconditions
To run this project you need to install Git, Java JDK and Maven:
- The lastest version of GIT you can find [here](https://git-scm.com/downloads)
- The lastest version of JDK you can find [here](https://www.oracle.com/br/java/technologies/downloads/) 
- The lastest version of Maven you can find [here](https://maven.apache.org/download.cgi)

After the installation,  Maven must be configured in the application path.

## Installation

After the Preconditions step, clone this project to a folder of your preference:

```bash
git clone https://github.com/Pablogarcia10/PabloAutomationTest.git
```

First you need to install the project dependecies:

```bash
mvn clean install
```

Run the tests of the project without open an IDE:
```bash
mvn test
```

## About this Project
This is an automation testing project of the website [Correios](https://www.correios.com.br). For this project I used some tools and frameworks like
- Java
- Cucumber
- Selenium Webdriver
- Junit and TestNG
- Maven
- BDD with Gherkin
